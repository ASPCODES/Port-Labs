// Lab: Basic clickjacking with CSRF token protection

This lab contains login functionality and a delete account button that is protected by a CSRF token. A user will click on elements that display the word "click" on a decoy website.

To solve the lab, craft some HTML that frames the account page and fools the user into deleting their account. The lab is solved when the account is deleted.

// Solution

Clickjacking is an interface-based attack in which a user is tricked into clicking on actionable content on a hidden website by clicking on some other content in a decoy website. We have looked at clickjacking as a self-contained attack.


we user <iframe> HTML element to solve the lab 

The <iframe> HTML element represents a nested browsing context, embedding another HTML page into the current one.
