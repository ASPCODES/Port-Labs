// Another common technique is to use a form element along with an element such as input to clobber DOM properties. 
For example, clobbering the attributes property enables you to bypass client-side filters that use it in their logic. Although the filter will enumerate the attributes property, it will not actually remove any attributes because the property has been clobbered with a DOM node. As a result, you will be able to inject malicious attributes that would normally be filtered out.

For example:

<form onclick=alert(1)><input id=attributes>Click me
